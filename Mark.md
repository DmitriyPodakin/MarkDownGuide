## Форматирование текста

Для жирного шрифта используем 2звёздочки (**) : **ghbvth**

*Курсив* - * для него используем по 1 звездочке по бокам


## Списки

Для нумерованого списка нужно писать цифры :
1. Первое
2. Второе
3. Третье
   
Для ненумерованого нужно писать * :
* Элемент
* Элемент
